Class: Survey

Survey

new Survey(_jsonSurvey)

Survey constructor. Takes a json survey and returns an internal survey object. It contains the fields:
  • filename The path of the source file used to generate this survey; may be empty.
  • topLevelBlocks The list of top level blocks in this survey.
  • breakoff Boolean indicating whether breakoff is permitted for this survey.
  • questions list of all questions. This gets wrapped in an Immutable.List.
Parameters:
Name Type Description
_jsonSurvey json the input survey, typically generated by another program.
